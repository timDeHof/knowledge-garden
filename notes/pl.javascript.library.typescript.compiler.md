# configuring the TypeScript compiler

```typescript
tsc-- - init
```

this will create a `tsconfig.json` file

## common configuration settings

    - "target" : "es2016"
    - "rootdir" : "./" switch to "./src"
    - "outdir" : "./" switch to "./dist"
    - "removeComments" : true
    - "noEmitOnError" : true
