TypeScript is a syntactic superset of JavaScript which adds static typing.

This basically means that TypeScript adds syntax on top of JavaScript, allowing developers to add
types.

## Benefits

- Static typing
- code completion
- Refactoring
- Shorthand notations

## Drawbacks

- Compilation
- discipline in coding
